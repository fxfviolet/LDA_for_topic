# LDA（Latent Dirichlet Allocation）提取主题词
#### LDA（Latent Dirichlet Allocation）隐含狄利克雷分布，是一种主题模型算法被广泛使用。LDA能够将文本集合转化为不同概率的主题集合，LDA利用统计手段对主题词汇进行处理，是一种词袋（bag-of-words）方法。
